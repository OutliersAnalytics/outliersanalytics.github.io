Sobre

Dados eleitorais brasileiros com clareza e precisão

Nossa missão

A Outliers Analytics nasceu da necessidade de transformar dados eleitorais brasileiros em informação clara e acessível. Acreditamos que decisões melhores começam com dados melhores — e que a inteligência eleitoral deve estar disponível para todos, não apenas para especialistas.

Nossa plataforma combina dados públicos do TSE com ferramentas modernas de visualização e análise. De votos válidos a abstenção, passando por comparecimento e resultados detalhados por seção eleitoral, oferecemos dashboards que facilitam a compreensão do cenário político brasileiro.

Nossos pacotes

Desenvolvemos ferramentas de código aberto em Python para acesso programático a dados eleitorais e governamentais. O pacote Dados Abertos Brasil (DAB) oferece uma interface unificada para APIs do governo, enquanto o cellmate facilita a criação de planilhas Excel estilizadas para exportação de resultados.
