Dados eleitorais abertos transformam a forma como entendemos a democracia. A transparência é a base para decisões mais claras e informadas.

Código pré-formatado

import dados_abertos_brasil as dab

# Buscar dados de candidatos
candidatos = dab.buscar_candidatos(
    ano=2024,
    cargo='vereador'
)

print(f'{len(candidatos)} candidatos encontrados')

Esquerda & Direita

Nossa plataforma de inteligência eleitoral transforma dados públicos em visualizações claras e acionáveis. Acompanhe resultados eleitorais, compare candidatos e analise tendências com dashboards interativos. Combinamos rigor estatístico com design intuitivo para que você tome decisões baseadas em dados, não em suposições. Dados abertos do TSE processados e apresentados com clareza.

Nossos pacotes Python de código aberto facilitam o acesso programático a dados eleitorais brasileiros. O Dados Abertos Brasil (DAB) unifica APIs governamentais em uma interface simples e consistente. Já o cellmate permite criar planilhas Excel estilizadas para exportação profissional de resultados. Ferramentas desenvolvidas para cientistas de dados, jornalistas e analistas políticos.
