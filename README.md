# Outliers Analytics

🌐 https://outliersanalytics.github.io

Website da Outliers Analytics, construído com [Jekyll](https://jekyllrb.com/) e o tema [Agency Jekyll Theme](https://github.com/raviriley/agency-jekyll-theme).

## Desenvolvimento Local

Para rodar localmente:

1. Instale Ruby, Bundler e Jekyll
2. Execute:

```bash
bundle install
bundle exec jekyll serve
```

## Deploy

O deploy é automático via GitHub Pages a partir da branch `main`.
